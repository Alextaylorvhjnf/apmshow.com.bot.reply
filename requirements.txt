# پروژه چت‌بات هوشمند سمت‌کلاینت
# تمام نیازمندی‌ها از طریق CDN بارگذاری می‌شوند

CDN_Dependencies:
- @mlc-ai/web-llm@0.2.34
- font-awesome@6.4.0
- vazirmatn-font@latest

Browser_Requirements:
- WebGPU Support (Chrome 113+, Edge 113+)
- JavaScript ES6+
- LocalStorage enabled
- Minimum 2GB RAM recommended for AI model

Optional_Features:
- WebAssembly support (for better performance)
- IndexedDB (for caching model files)

Deployment:
- Static hosting (GitHub Pages, Netlify, Vercel, Render, Railway)
- No backend required
- HTTPS recommended for service worker caching

Development:
- Modern browser for testing
- Local web server for testing (python -m http.server 8000)
- Basic understanding of JavaScript

Note: This is a client-side only application. All processing happens in the browser.
